개인정보처리방침 Privacy Policy

서바이벌 니혼고 (Survival Nihongo)

1. 개요 Overview

"서바이벌 니혼고(Survival Nihongo)"는 일본어 학습 게임 앱입니다. 본 앱은 사용자의 개인정보를 수집하지 않으며, 개인정보 보호를 중요하게 생각합니다. 본 방침은 앱의 데이터 처리에 관한 사항을 안내합니다.

2. 개인정보 수집 Data Collection

본 앱은 어떠한 개인정보도 수집하지 않습니다.

  • 회원가입 및 로그인이 필요하지 않습니다.
  • 이름, 이메일, 전화번호 등 개인 식별 정보를 요구하지 않습니다.
  • 위치 정보를 수집하지 않습니다.
  • 기기 정보를 수집하거나 외부로 전송하지 않습니다.

3. 외부 API 사용 Third-Party API Usage

본 앱은 학습 기능을 위해 OpenAI API를 사용합니다. 이 과정에서 다음과 같은 데이터만 전송됩니다:

  • 학습 중인 일본어 텍스트 (힌트 생성 용도)
  • 일본어 텍스트 (음성 합성(TTS) 용도)

사용자의 개인정보는 일체 전송되지 않으며, 전송되는 데이터는 앱 내 학습 콘텐츠에 한정됩니다. OpenAI의 개인정보처리방침은 https://openai.com/privacy에서 확인하실 수 있습니다.

4. 광고 및 분석 도구 Advertising & Analytics

본 앱은 광고를 포함하지 않습니다. 또한 사용자 행동을 추적하거나 분석하는 도구(애널리틱스)를 사용하지 않습니다.

5. 데이터 저장 Data Storage

학습 진행 상황 등의 데이터는 사용자의 기기 내에만 저장되며, 외부 서버로 전송되거나 공유되지 않습니다.

6. 아동 보호 Children's Privacy

본 앱은 개인정보를 수집하지 않으므로 아동의 개인정보 역시 수집하지 않습니다. 모든 연령대의 사용자가 안전하게 이용할 수 있습니다.

7. 개인정보처리방침 변경 Changes to This Policy

본 방침이 변경될 경우 이 페이지를 통해 공지합니다. 변경 사항은 페이지에 게시된 시점부터 효력이 발생합니다.
