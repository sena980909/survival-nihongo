========================================
Survival Nihongo - 다음 개선 계획
========================================

[1] 직접 입력 모드 (자유 응답)
- 선택지 대신 직접 일본어를 타이핑하거나 음성으로 답하는 모드
- OpenAI API로 자유 입력을 평가하고 피드백 제공
- 음성 인식(Web Speech API 또는 Whisper)으로 발음 연습
- 선택지 모드(초급)와 자유 입력 모드(중급)를 사용자가 선택

[2] 분기형 대화
- best/acceptable/poor 선택에 따라 NPC 반응이 달라지는 분기 구현
- poor를 고르면 NPC가 다시 물어보거나 다른 상황으로 전개
- 리플레이 가치 향상 — 같은 시나리오를 다른 루트로 경험

[3] 자동 복습 시스템
- 북마크 없이도 대화 중 나온 모든 표현을 자동 복습 풀에 저장
- 시나리오별 핵심 표현 퀴즈 (keyExpressions 기반)
- 에빙하우스 망각곡선 기반 간격 반복 (spaced repetition)
- "오늘의 복습" 알림 (PWA push notification)

[4] 오프라인 TTS
- Web Speech API의 일본어 음성을 fallback으로 사용
- 또는 주요 표현의 TTS 음성을 미리 캐싱 (service worker)
- 비행기/지하철 등 오프라인 환경에서도 발음 확인 가능

[5] 시각적 몰입감 향상
- 픽셀아트 캐릭터를 대화 화면에 NPC로 표시
- 시나리오별 배경 이미지 (공항, 택시, 편의점, 카페 등)
- NPC 감정에 따라 캐릭터 표정 변화 (neutral/happy/confused)
- 이모지(😐😊) 대신 실제 일러스트로 교체

[6] 깊이 있는 학습 분석
- 자주 틀리는 패턴 분석 (문법별, 표현별)
- 약한 문법 포인트 추적 및 추천
- 학습 시간 추적 (일별/주별 그래프)
- 가장 많이 복습한 표현 순위

[7] 여행 동선 스토리 모드
- "공항 → 스카이라이너 → 호텔 → 편의점 → 관광지" 순서로 연결
- 여행 1일차, 2일차... 형태의 스토리 흐름
- 이전 시나리오에서 배운 표현이 다음 시나리오에 등장 (연계 학습)

[8] 추가 시나리오 후보
- 온천/료칸 매너
- 신칸센 탑승
- 이자카야 추가 주문/계산
- 길에서 사진 부탁하기
- 분실물 신고 (교번/경찰)
- 공항 면세점
